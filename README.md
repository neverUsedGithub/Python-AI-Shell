# Python AI Shell

An AI designed to help you write shell commands, using `mistral-7b-instruct-v0.1` from [gpt4all](https://gpt4all.io).

# Required PC specs
- 8 GB ram
- 3.83 GB of free space

# Getting started
### Firstly install the required packages.
`pip install -r requirements.txt`
### Then just run it.
`python main.py`
